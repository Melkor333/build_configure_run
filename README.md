# Build - Configure - Run

These are the typical 3 steps in:
- code deployment
- systems management
- container management
- service management
- ...


Wait, are they all the same? Maybe! This is an experiment to see how many tools with vastly different targets we can map to the exactly same (sub)graph.

Can we draw conclusions from one tool in the subgraph that apply to most - or even *all* - tools that map to the same nodes?
